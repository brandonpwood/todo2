What is Todo 2?

Todo 2 is a revolutionary todo app that will fundamentally synergize the way big data analytics controls productivity flow. By operating in a lightweight, mobile-first bootstrap envirnonment, this single page web app was build with the philosophy that it would not only drive progress in todo app design, but all app kind.

Why now?

Why did we think now was the time to release todo 2? We are fortunate to be living in a time where machine learning, big data analytics, and artificial intelligence are able to propel productive synergies through stochastic heuristics and hyper-dimmensional analysis. The only real question is: What took us so long?
